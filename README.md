# MyPortfolio
LINE公式アカウント上で動作するチャットBotです。

# 主な機能
・商品情報表示

・商品発注手続き

・買い物かご

# 環境
**フロントエンド（プログラムは未掲載）**

Google apps Script(javascriptベース)

LINE Front-end Framework（LIFF）


**バックエンド**

Node.js


**インフラ**

firebase

Googleスプレッドシート


**API**

LINE Messaging API
